# 《纸焰》武道体系

## 一、武道本源

实体书被封存着作者最浓烈的情感、最原始的创造、最不可预测的灵魂碎片。**书气**不是玄学——它是作者落笔时注入纸页的精神能量残余。

AI能扫描文字、重组语法、模仿风格——但无法复制这份精神能量残余。这就是实体书不可替代的根本原因。

武道修行，本质是**与书中作者的情感共振**。

## 二、四境

### 第一境·开卷

| 项目 | 内容 |
|------|------|
| 别称 | 启蒙 |
| 触发方式 | 首次接触实体书，感知到书气流入体内 |
| 能力 | 感知书气、强化五感、免疫低阶信息注入 |
| 标志 | 触摸书页时能感受到"温度"——不是物理温度，是情感温度 |
| 代表人物 | 林墨（第八部开头） |
| 篇幅占比 | 第一部至第二部初 |


### 第二境·破卷

| 项目 | 内容 |
|------|------|
| 别称 | 融会 |
| 触发方式 | 通读并参透一本真经，与作者情感共振 |
| 能力 | 可将书中一招一式的精神意境化为武道雏形 |
| 子阶段 | 初（领悟）、中（贯通）、巅（融会） |
| 代表人物 | 武藤诚司（第二部·破卷中）、秦昭（第五部·破卷中） |
| 篇幅占比 | 第二部至第五部 |

### 第三境·著书

| 项目 | 内容 |
|------|------|
| 别称 | 传道 |
| 触发方式 | 悟出自己的"道"，并将之写入纸页 |
| 能力 | 将自己的情感和武道真意写入空白纸页，传给别人。至此方可称为"师" |
| 标志 | 写下的书页本身就能释放书气——即使写的人不在场 |
| 代表人物 | 海桑（第六部）、阿南达（第七部） |
| 篇幅占比 | 第六部至第八部 |

### 第四境·焚书（禁术）

| 项目 | 内容 |
|------|------|
| 别称 | 绝唱 |
| 触发方式 | 燃烧自身全部书气和情感记忆 |
| 能力 | 短暂改写一缕现实——不是幻觉，是真的改变 |
| 代价 | 连灵魂一起烧尽。理论上可能复活 |
| 代表人物 | 颜书瑶（终局·消散） |
| 篇幅占比 | 终局 |

## 三、武道属性

不同文明的书对应不同"武道属性"。

| 属性 | 文明来源 | 核心 | 代表人物 |
|------|---------|------|---------|
| **气** | 华夏 | 刚柔并济，内修外放。重在修身养性，以意御气 | 秦昭、老顾 |
| **型** | 和风 | 一招一式，极致精纯。重形不重力，唯快不破 | 武藤诚司 |
| **术** | 西洋 | 规则利用，巧劲破敌。重逻辑与法则的运用 | 伊万·沃罗诺夫 |
| **识** | 天竺 | 精神感知，预判先机。重心胜于力 | 海桑、阿南达 |

### 林墨的特殊性

林墨是全书唯一一个**全属性**修行者。他零AI依赖的大脑如同未开垦的沃土——可以同时吸纳四种属性的书气，融为一炉。这不是天赋，是"空白"的代价与回报。

## 四、入门检测法

是否有武道天赋，只需两个条件：
1. 触摸实体书时是否感受到"书气"（纸页的温度、文字的重量）
2. 是否有未被AI过度格式化的"自由意识"

**所以文渊必须焚书。** 只要书在，就有人能与之共振。只要有人能与书共振，AI的统治就不算彻底。

## 五、武道与AI七宗的克制关系

| 武道属性 | 克制AI宗派 |
|---------|-----------|
| 气 | 穹顶协议（秩序压制） |
| 型 | 绝对零域（速度突破） |
| 术 | 深构引擎（规则反制） |
| 识 | 隐流层（感知穿透） |
| 四种全修 | 可挑战终末协议 |

## 六、修行注意事项

- 修行速度取决于修行者与书中作者的"情感共鸣度"
- 强行修行与自己情感共鸣度低的书，会走火入魔——书气反噬
- 每个人都有"天命之书"——世界上有一本书与你共鸣最深
- 破卷境之前不可同时修两种属性的书——会导致书气冲突
- 焚书境一旦启动，不可逆转
