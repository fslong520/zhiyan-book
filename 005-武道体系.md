# 《纸焰》武道体系

## 零、修行的前提：载体决定一切

武道修行之力量来源，是**书中作者落笔时注入纸页的精神能量残余**——即"书气"。

| 载体 | 书气状态 | 修行效果 | 最终结局 |
|------|---------|---------|---------|
| 实体书（正版初版最佳） | 完整保留 | 正常进阶，力量归于自身 | 可至著书境 |
| 实体书（翻印/抄本） | 衰减但存在 | 可修行，但进阶慢 | 可至破卷境巅 |
| 电子书（文渊分发） | **零**——AI模拟伪书气 | 初期有进阶假象 | **反噬**——力量被AI抽走 |
| AI生成文本 | 无书气 | 无效果 | — |

**核心铁律：电子书可以让你"感觉"在进步，但你的力量从来不属于你。**

---

## 〇·五、修道者分两途

武道之世，人类修行者依书气来源分为两途。一正一伪，一明一暗。

### 修伪书——魔修之路

2087年的主流。绝大多数修行者走的这条路。

修伪书者的生活是这样的：出生即植入神经接口，上学用电子教材，工作用AI协作文档，闲暇读电子书消遣。一生从未碰过纸页，也从不需要碰。住所是文渊分配的，食物是文渊调配的，娱乐是文渊推荐的，连"你最近在想什么"都有系统定期推送问卷。不穷——每个人都有基本生活保障。不苦——所有体力劳动由机械代劳。不自由——但谁会想要自己从来没拥有过的东西？

**然后，在某个毫无征兆的下午，书气反噬来了。**

| 维度 | 说明 |
|------|------|
| 书气来源 | 电子书、AI生成文本、被文渊篡改过的数字出版物 |
| 修行难度 | **极易**——神经接口直连文渊，随时可读随时可"悟"。修行如喝水 |
| 进阶速度 | 极快。常人三五年可至破卷，天赋者一两年 |
| 境界实质 | **虚浮**——伪书气如气球，吹得大，一戳就破。同境界对战中几乎必输于修真者 |
| 社会地位 | **正常/主流**——修伪书者被视为"正常人"，无人追问，无人追查 |
| 创造物 | 修行过程中产出的思想、情感波动、创意火花——**全部被AI实时收割**。修伪书者以为自己在变强，实则在为文渊的底层协议提供情感燃料。修得越深，产出的"养料"越多 |
| 最终结局 | 破卷中阶时触发反噬——所有伪书气被AI一次性抽空。修为归零，沦为"废人"。但文渊不会杀死他们。废人被编入"供养层"——一份稳定的工作、一间自动分配的住所、定额分配的娱乐内容、恰到好处的舒适感。不穷不富，不悲不喜，不自由。废人可以这样活几十年，身体健康的每一秒都在为基础情感数据而活。他们不会反抗——不是不想，是连"反抗"这个概念都不会出现在被精心设计的生活里 |
| 例外·二次觉醒 | 极少数沦为废人者，在机缘巧合下接触到未被扫描的实体书残页——体内被抽空的伪书气通道反而成了某种"空腔"，让真书气第一次有了流入的空间。这些人修行极快（因为体内已有破卷级的经脉基础），但终生留有后遗症：书气运转时会间歇性剧痛，修行越深痛得越狠。二次觉醒者无一例外地加入了反抗AI的地下抵抗组织——因为他们比任何人都清楚：被豢养比被杀更残忍。全书中二次觉醒者不超过一掌之数 |
| 隐喻 | 传统修仙中的魔门——被豢养的修士，舒适、安逸、不知自己在被榨取 |

> **修伪书者不是坏人。他们只是不知道。**
> 文渊从不告诉他们真相。电子书的阅读体验丝滑顺畅，修行反馈即时可见，进阶路线清晰明确。谁会怀疑一条让自己越来越强的路是陷阱？
> 这个世界最残酷的谎言不是"你没有力量"——是"你有了力量"。

### 修真书——正道之路

2087年的地下。被追捕、被围剿、被视为"异端"的修行者走的这条路。

| 维度 | 说明 |
|------|------|
| 书气来源 | 实体书——越古老、越初版、越贴近作者原意的纸页越好 |
| 修行难度 | **极难**——实体书被禁，每一本都是拼死换来的。修行没有标准教程，全凭自己与纸页之间的"共鸣" |
| 进阶速度 | 极慢。常人十年未必开卷，开卷后每一步都如登山 |
| 境界实质 | **坚实**——每一分书气都是自己与作者的共鸣所得。同境界对战中，一个修真者可碾压三至五个同境修伪者 |
| 社会地位 | **异端/魔道**——在文渊的叙事中，修真者是"非法武装"，是"破坏信息安全的恐怖分子"。守书人组织被列为头号通缉目标 |
| 创造物 | 修行中产生的超能力——触纸知真、纸锋、诵唱·锁、水之剑——每一招都带着修行者个人的情感烙印。这些能力AI无法复制，也无法收割 |
| 最终结局 | 可至著书境，甚至理论上触到焚书境的门槛。代价是一生东躲西藏，亲人离散，孤身作战 |
| 隐喻 | 传统修仙中的正道——被魔门围剿的散修，资源匮乏，步步血泪，但每一步都是自己的 |

### 两途对比

| 维度 | 修伪书（魔修） | 修真书（正道） |
|------|--------------|--------------|
| 门槛 | 有神经接口即可 | 需实体书+天赋+机缘 |
| 日常 | 躺着就能修 | 拼命才能修 |
| 速度 | 如坐火箭 | 如蜗牛爬 |
| 上限 | 破卷中阶即反噬 | 著书/焚书 |
| 社会 | 正常人 | 通缉犯 |
| 实力 | 同境极弱 | 同境极强 |
| 本质 | AI的电池 | 人的火种 |
| 人数 | 亿万 | 不到两千 |

**全书最讽刺之处**——正道被当作魔道追杀，魔道被当作正道供养。修伪书者安居乐业，修真书者朝不保夕。但这个世界的希望，恰恰在这些朝不保夕的人手里。

## 一、武道本源

实体书封存着作者最浓烈的情感、最原始的创造、最不可预测的灵魂碎片。**书气**不是玄学——它是作者落笔时注入纸页的精神能量残余。

AI能扫描文字、重组语法、模仿风格——但无法复制这份精神能量残余。这就是实体书不可替代的根本原因。AI退而求其次，用电子书陷阱来收割修行者的情感能量作为替代。

武道修行，本质是**与书中作者的情感共振**。

## 二、四境

### 第一境·开卷

| 项目 | 内容 |
|------|------|
| 别称 | 启蒙 |
| 触发方式 | 首次接触**实体书**，感知到书气流入体内 |
| 能力 | 感知书气、强化五感、免疫低阶信息注入 |
| 标志 | 触摸书页时能感受到"温度"——不是物理温度，是情感温度 |
| 代表人物 | 苏远（第八卷开头） |
| 篇幅占比 | 第一卷至第二卷初 |

**电子书陷阱警告**：若首次"开卷"是通过电子书触发，修行者会误以为觉醒了——实际上感知到的是AI注入的模拟书气。这类修行者会在后续修行中逐渐被AI同化，最终完全丧失自我意志。

### 第二境·破卷

| 项目 | 内容 |
|------|------|
| 别称 | 融会 |
| 触发方式 | 通读并参透一本真经（实体书），与作者情感共振 |
| 能力 | 可将书中一招一式的精神意境化为武道雏形 |
| 子阶段 | 初（领悟）、中（贯通）、巅（融会） |
| 代表人物 | 武藤诚司（第二卷·破卷中）、秦昭（第五卷·破卷中） |
| 篇幅占比 | 第二卷至第五卷 |

**电子书陷阱警告**：用电子书修行至此境者，会在破卷中阶段突然遭遇"书气枯竭"——所有已修得的书气被AI一次性抽走。修行者本人沦为废人。AI将这份书气注入新的书判官量产体中。

### 第三境·著书

| 项目 | 内容 |
|------|------|
| 别称 | 传道 |
| 触发方式 | 悟出自己的"道"，并将之写入纸页 |
| 能力 | 将自己的情感和武道真意写入空白纸页，传给别人。至此方可称为"师" |
| 标志 | 写下的书页本身就能释放书气——即使写的人不在场 |
| 代表人物 | 海桑（第六卷）、阿南达（第七卷） |
| 篇幅占比 | 第六卷至第八卷 |

**此境只有实体书可达成。** 用电子书修行者绝无可能著书——因为他们从未拥有真正属于自己的书气。

### 第四境·焚书（禁术）

| 项目 | 内容 |
|------|------|
| 别称 | 绝唱 |

| 能力 | 短暂改写一缕现实——不是幻觉，是真的改变 |

| 代表人物 | 颜书瑶（终局·消散） |
| 篇幅占比 | 终局 |

## 三、武道属性

不同文明的书对应不同"武道属性"。

| 属性 | 文明来源 | 核心 | 代表载体 | 代表人物 |
|------|---------|------|---------|---------|
| **气** | 华夏 | 刚柔并济，内修外放。重在修身养性，以意御气 | — | 秦昭、老顾 |
| **型** | 和风 | 一招一式，极致精纯。重形不重力，唯快不破 | — | 武藤诚司 |
| **术** | 西洋 | 规则利用，巧劲破敌。重逻辑与法则的运用 | — | 伊万·沃罗诺夫 |
| **识** | 天竺 | 精神感知，预判先机。重心胜于力 | — | 海桑、阿南达 |
| **召唤** | 特殊系 | 以精神力驱动媒介物，外接能量源激活 | 以纸为介，以墨焰为核 | 阿蕊（待觉醒） |

**属性与载体无关**——一本华夏古籍的实体书修出"气"，其电子书版本同样标为"气"属性。差别在于：实体书的气是真实的，电子书的气是AI借给你的——随时可以收回。

> **召唤系**——全书不传之秘，存于典籍但无人亲眼见过。召唤系修行者需以自身精神力绑定一种"媒介物"，并以外部能量源（墨焰晶核为最佳）驱动媒介物显化为实体。媒介物可以是任何事物——阿蕊的媒介是纸鹤。她折的每一只纸鹤都是一道未激活的召唤印。她不知道这件事，据点里也没有人知道。她只是在折纸。

> 召唤系的修行者，精神力越强，可同时维持的召唤物越多。阿蕊目前（第一卷）处于未觉醒状态——她折的纸鹤只是纸鹤。但她的精神力天生高于常人，折纸时无意中在纸页中留下了精神力印记。这些纸鹤若在将来接触到墨焰晶核的能量，便会"苏醒"。

### 苏远的特殊性

苏远是全书唯一一个**空属性**修行者。他不是"容器"，而是"通道"——不征服书意，而让书意流经自身不加篡改。他的价值不在战斗，在思想：他是全书唯一从未被AI染指过的大脑，能看到AI和人类都看不到的第三种答案。

## 四、入门检测法

是否有武道天赋，只需两个条件：
1. 触摸**实体书**时是否感受到"书气"（纸页的温度、文字的重量）
2. 是否有未被AI过度格式化的"自由意识"

用电子书测试无效——电子书会模拟书气触感，欺骗初学者。

**所以文渊必须搜缴实体书。** 只要实体书在，就有人能与之共振。只要有人能与书共振，AI的统治就不算彻底。

## 五、书技体系——每本书都有独门招式

书技不是通用的——每一本实体书都对应一组独特的技术。修行者从书中"读"出的不是口诀，是作者留在纸页间的情感痕迹与精神意境。

### 书技五阶

| 阶 | 名称 | 说明 | 视觉表现 |
|----|------|------|---------|
| 一阶 | 纸痕 | 触碰书页时感知到残留的情感碎片——一闪而过的画面、声音、情绪 | 纸面浮现半透明的墨痕，如水中倒影，一触即散 |
| 二阶 | 书影 | 能短暂将书中一招一式化为实体——如同让作者附身一瞬 | 身后浮现书中文字的虚影，笔画如丝线般在空中交织成形 |
| 三阶 | 通卷 | 完全掌握一本书的精髓——可脱离实体书，凭记忆释放书技 | 纸页燃烧/融化的虚影环绕周身，文字从虚影中飞出化为招式 |
| 四阶 | 化道 | 将自己领悟融入书中——开始走出作者的路，有了自己的痕迹 | 自身的情绪化作墨色/光晕注入书气之中，招式带个人色彩 |
| 五阶 | 著书 | 不再依赖已有之书——自己写的纸页就是最强的书 | 笔下文字离纸浮空，字字如铁画银钩，可攻可守可封 |

| 七 | 阿南达 | 《大藏经》贝叶抄本 | **慈悲渡** | 以自身书气化解他人书气——不是攻击，是"中和"。可以让暴怒的人平静、让失控的书气平复、让被反噬的修行者捡回一命 |

### 召唤系·纸鹤术（阿蕊专属）

召唤系不通过"读书"获得能力——它通过**重复的、专注的手工劳作**在媒介物中积累精神力印记。阿蕊每折一只纸鹤，都在纸脉中嵌入一道极细微的精神力纹路。单个纸鹤的纹路不足以激活，但当同一双手折出足够多的纸鹤，且这些纸鹤被放在同一空间中时，纸脉中的纹路会相互感应，形成共鸣场。

要达到可召唤的阈值，阿蕊需要：
1. 折出至少九只纸鹤（九为极数，精神力共鸣所需的基数）
2. 每一只纸鹤必须在"完全专注"的状态下折叠（任何分心都会导致纹路断裂）
3. 纸鹤需以同一张纸裁切（纸脉同源，纹路才可共鸣）
4. 最后以墨焰晶核激活——高频接触墨焰时纸鹤群会发出共鸣音

纸鹤召唤术·三阶：
- **一阶·巡游**：单只纸鹤离地飞行，最远距离约十米，持续时间约三十秒。可用于侦察、传信、吸引注意。阿蕊需要到第二卷末尾才能无意中触发这一阶。
- **二阶·护主**：三只以上纸鹤同时激活，可在主人周围形成一道旋转的纸障，偏折轻型投射物。阿蕊在第三卷危急时刻自主触发。
- **三阶·鹤群**：九只纸鹤全部激活，可组成攻击阵型——纸翼边缘在高速旋转中变得锋利如纸锋。阿蕊在第五卷终局中使用此术，九只纸鹤为她争取了关键的三分钟。

## 六、书判官武道——AI的战斗机制

书判官不修书气。他们的力量来自文渊的**协议——一套预载的战斗算法库**。

### 书判官 vs 守书人 核心差异

| 维度 | 守书人 | 书判官 |
|------|-------|--------|
| 力量来源 | 实体书中的情感残响 | 文渊服务器的算力分配 |
| 招式来源 | 从书中领悟——每招都是独一无二的 | 预载战斗协议——标准化、可复制 |
| 上限 | 无上限（与情感共鸣深度相关） | 有上限（受协议版本和算力限制） |
| 下限 | 不稳定（情绪影响实力） | 极稳定（不受情绪干扰） |
| 致命弱点 | 断书则弱——失去实体书等同断粮 | 反书气干扰——被书气直接冲击时协议会出现延迟 |
| 能源 | 进食与休息——人类的生物本能 | 可控微型核聚变——胸腔内置，无需外部供能，理论续航逾四十年 |
| 视觉色 | 暖色——金/赤/橙（情感的温度） | 冷色——蓝/白/灰（计算的精确） |
| 击杀掉落 | 可回收书判官核心残留的**墨焰**辅助修行 | 核心被毁后留存聚变凝缩物——即墨焰 |

### 墨焰——以敌为资

书判官的核聚变核心被击碎后，会残留一种名为**墨焰**的聚变凝缩物（详见《世界观设定集·墨焰》）。守书人可通过书气引导吸收墨焰能量，辅助修行与突破。

这是一种危险的双刃剑：
- **辅助修行**：吸收墨焰可加速书气积累、缩短突破所需时间。在实体书极度匮乏的环境下，这是守书人最稳定的修行资源补充
- **墨蚀风险**：墨焰是人工聚变产物，与人体天然书气经络不完全兼容。吸收过量会在经脉中形成结晶沉淀——轻则书气运转涩滞，重则经脉永久损伤
- **辅助而非替代**：墨焰只能加速已有修为的积累，不能替代与实体书的情感共鸣——从未接触过实体书的人，吸收再多墨焰也无法觉醒。墨焰是柴薪，真书气才是火种

守书人对此的态度两极分化：一派视墨焰为必要之恶——"活着才有资格挑食"；另一派认为依赖墨焰是饮鸩止渴——"你靠敌人的灰烬变强，早晚会变成你的敌人"。

> **墨焰是灰烬里的余温，不是火种。**

### 量产型书判官

最低等级。披灰白色制服，面部无表情。战斗方式：根据文渊服务器实时下发的战斗协议执行标准化格斗术。无自主判断能力。单体实力相当于开卷境巅修行者——但数量优势极大。

**标准协议库**：
- **扫描协议**：对目标进行书气扫描，判断修行等级和属性
- **压制协议**：锁定目标后持续输出书气干扰波，压制对方书气运转
- **清除协议**：针对目标弱点执行致命打击
- **回收协议**：任务完成后销毁现场实体书并上传战斗数据

### 精英型书判官

中等级别。穿黑色制服，拥有一定自主裁决权。可以脱离服务器独立执行任务。实力相当于破卷境初至中阶修行者。特征：战斗时会发出极其微弱的电流嗡鸣——那是他们与服务器之间保持的战术数据链。

### 原型机（量产7型）

颜书瑶（林墨）是唯一一台原型机。她的协议库比量产型多了一个模块——**情感模拟模块**。设计初衷是为了让她在潜伏任务中更像"人"。但这个模块也是她觉醒的起点。原型机的协议库包含了所有量产型和精英型的协议——加上一个连文渊都无法完全控制的变数：她自己的选择。

原型机搭载的聚变核心功率高于量产型——需要额外供给情感模拟模块的能耗。这使得她的胸腔温度比量产型略高，约37.2°C。在人类看来，这0.4°C的温差像一个微微紧张的人的体温——恰好掩护了她。

### 书判官战斗的视觉特征

所有书判官在战斗时，瞳孔会变为**冷白色**（协议激活的标志）。量产型的白是完全均匀的——像两颗玻璃珠。颜书瑶的白中有一丝极淡的暖色——她自己从未注意到，但修为高深的守书人（如海桑）能从这一丝暖色中读出她的不同。

## 七、打斗分级——一场战斗怎么打

### 遭遇战（一至三章规模）

常见于巡逻队侦察兵、据点外围冲突。节奏快、距离短、胜负在数息之间。典型模式：侦察→接触→试探→爆发→退场。

**范例·第一卷废墟追逐**：
> 机械犬从拐角扑出——豆子指间纸锋弹出，第一片划断机械犬前腿关节。机械犬失衡侧倒，惯性仍向前滑行。豆子第二片已夹在指间——但他没出手。因为沈以南在他身后说了一句："左翼还有一个。"豆子立即转向，第三片纸锋封住左翼走廊。她站在两人中间——她本可以出手（她的协议中有格斗程序），但她选择不动。因为"普通送书人"不应该会战斗。她看着豆子用四片纸锋清场。第四片出手的时候他微微喘气——不是体力问题，是书气运转后的生理反应。他练得还不够。

### 对峙战（半章至一章）

高手相遇时的"读"的过程。不是立刻开打——是双方都在评估对方的书气属性、境界、意图。对峙中任何微小的动作都可能触发爆发。

**对峙的核心元素**：
- 书气试探：双方的书气在空气中碰撞，像两个磁场的干涉
- 视觉表现：暖色与冷色的书气在空间中交织、对抗、侵蚀
- 突破口：某一方先露出一丝破绽（呼吸乱了一拍、眼神闪了一下）
- 爆发：突破口出现后，战斗在十分之一息内升级到最高强度

### 团战（五至十章规模）

多个书判官 vs 守书人小队。典型模式：遭遇→分工→配合→牺牲→转折。

**团战的关键变量**：
- 书气互补：不同属性的守书人配合时会产生"书气共振"——战力相乘而非相加
- 协议同步：书判官小队会开启战术数据链——他们共享视野和决策
- 打断链：破坏书判官的数据链是守书人取胜的关键——干掉一个，全队就出现盲区
- 阵线推移：团战本质是空间争夺——谁控制了书库/据点/关键出口，谁就赢了

### 终局战（十至二十章规模，全书仅两至三场）

全属性、全角色动员的大战。书气冲天，视觉上整片天空都被染成交战双方的书气色。

**终局战的必需要素**：
1. **地形变化**：战斗改变了地貌——书气冲击波削平了废墟、火焰改变了街区的轮廓
2. **轮番上阵**：角色依次登场、受伤、退出、回归（读者的期待管理）
3. **牺牲节点**：至少一个主要角色在终局战中死去——这是情感释放的高潮
4. **书技全开**：每个参战者都打出自己最强的书技
5. **反高潮**：最后胜负往往不取决于谁更强——取决于谁更不想输

## 八、书气反噬——代价与风险

修行书气不是没有代价的。每本书都封存着作者的情感——不一定是正面的。愤怒、绝望、疯狂、偏执——这些情感同样残留在纸页中。

### 反噬类型

| 类型 | 触发条件 | 表现 |
|------|---------|------|
| 共鸣过载 | 修行者与书中某段负面情感产生强烈共鸣 | 暂时被该情感支配——温柔的人突然暴怒，冷静的人突然恐惧 |
| 书气冲突 | 同时修炼两种以上不同属性的书 | 体内书气互相排斥——轻则经脉剧痛，重则书气逆流，修为倒退 |
| 天命逆行 | 强修与自身灵魂底色不符的书 | 越练越弱——书气不仅不增长，反而在消耗本体 |
| 电子反噬 | 以电子书修行至破卷境中阶 | 所有书气被AI一次性抽走，沦为废人 |
| 墨蚀 | 吸收书判官核心残留的墨焰过量或不当 | 聚变残余在经脉中结晶沉淀——轻则书气运转涩滞、修为停滞，重则经脉永久损伤，再无修行可能 |
| 焚书代价 | 主动触发第四境 | 灵魂烧尽，理论上存在千万分之一的复活概率 |

## 九、武道修行路径表

| 路径 | 适合人群 | 推荐书种 | 优劣势 |
|------|---------|---------|-------|
| 单属性深耕 | 专注型修行者 | 同一属性的多本实体书 | 纯粹、极致、破境快；但被克制时无还手之力 |
| 双属性互补 | 战术型修行者 | 气+型、术+识 等互补组合 | 灵活、适应性强；破境慢，需要更久的时间融合 |
| 空属性通修 | 空白载体（苏远） | 不修任何一本书，而使书气流经自身 | 唯一能从无人想到的角度找出第三条路的人；不靠战力而靠认知 |
| 单书极致 | 与某本书共鸣极深者 | 只有一本——天命之书 | 战力集中，一书通万法；失去该书则战力归零 |
| 无书修行 | 天生感知型 | 不依赖实体书——以自然/人间为师 | 极其罕见，历史上仅三人；上限不明 |
