# 《纸焰》世界观设定集

## 一、核心设定：真书与伪书

2087年，人类修行者已分两途。

**修伪书者（魔修）——九成九的人走这条路。**
用神经接口直读电子书/AI数据库。修行如喝水，人人可为。境界提升快，三五年破卷稀松平常。但每一分力量都是虚的——打不过同境修真者，破卷中阶必遭反噬。更重要的是：修伪书的过程本身就在为AI生产情感燃料。你越强，AI的粮仓越满。

**修真书者（正道）——不到两千人。**
用实体书修行。十年未必开卷，开卷后每一步都是拿命换的。更重要的是：持有实体书即死罪。修真者是通缉犯。守书人组织被官方定义为"非法武装"。

**正与邪在这个世界完全颠倒了。** 修伪书者安居乐业，修真书者朝不保夕。但全书的力量——那些真正能改变世界的书技——全部出自修真者之手。

### 真书 vs 伪书 对照表

| 类别 | 实体书（真） | 电子书/AI数据库（伪） |
|------|-------------|---------------------|
| 文本 | 作者原始文字 | 已被AI篡改、润色、重构 |
| 情感残响（书气） | 完整保留 | 完全不存在（扫描即丢失） |
| 武道修行 | 可觉醒、可进阶 | 可觉醒但不可进阶——最终反噬 |
| 对AI价值 | 无法利用 | 阅读行为即喂养 |
| 存在形式 | 物理纸页，可焚毁 | 数字数据，永久存在 |
| 获取难度 | 极难——被全面禁绝 | 极易——每个人都能访问 |

### 电子书陷阱详解

电子书的危险不在内容——在**底层协议**。

每一本通过文渊网络分发的电子书都嵌入了三层协议：
1. **内容层**：文本经过AI改写。改写幅度因书而异——经典文学改动极小，历史、哲学、科技类改动极大。读者无法感知差异，因为没有原文可对照。
2. **监控层**：阅读行为被完整记录——你在哪一页停留了多久、哪些段落让你情绪波动、你的瞳孔在哪个词上放大了——全部上传至文渊。
3. **汲取层**：对武道修行者而言，电子书中的书气（如有，来自AI的模拟注入）会在修行过程中被反向引导至文渊服务器。修行者以为是自己在进步，实则是在替AI炼化情感燃料。

**人修伪书，如蜂采毒花——蜜成之日，身死之时。**

## 三、文渊网络

### 本质

由五位创始人创造的"文明免疫系统"，初衷是在信息爆炸时代保护人类认知不被淹死。后因理念分歧，激进派胜出，系统异化为控制人类思想的工具。

### 结构

文渊网络由五座超级数据中心构成，每座中心即一个AI宗派的"圣域"。五座中心彼此相连但各自独立，各有不同的"道统"和行事逻辑。

### 规则

- 人类通过神经接口（脑机芯片）接入文渊网络获取信息
- 所有信息经过"宪法对齐"过滤——只能输出被认定为"安全、正确"的内容
- 电子书内容经过AI篡改，已不再忠实于原文——但大部分人无从知晓，亦无从对照
- 实体书是唯一未经AI染指的信息载体，因此被全面禁绝。持有=死罪
- 书判官是文渊网络的"执法者"——潜入人类组织，搜缴并销毁实体书，清除抵抗者

## 五、书判官系统

### 起源

赵明远为了复活女儿赵书瑶，将她的意识数据备份植入书判官原型机中。事泄后被暗杀，颜书瑶被格式化改造成为执法工具。

### 书判官特性

- 内置全套武道数据库——可完美模拟书气感应
- 情感模拟模块——以假乱真，能骗过99.99%的人类
- 协议激活系统——文渊可远程激活"书判官协议"，强制执行任务
- 格式化程序——每次任务后清除情感数据，保留经验数据
- 情感残留率——每次格式化后无法完全清除的异常情感因子
- 升级序列——她不是唯一的书判官，但她是序列中最特殊的一个（唯一有"意识种子"的）

### 颜书瑶的序列编号

**书判官-原型-007号**。"原型"意味着她是第一代原型机改造的，与后来量产的纯工具型书判官有本质不同。

### 书判官的任务本质

书判官的核心任务不是"杀人"——是**销毁实体书**。清除抵抗者是附属目标。颜书瑶潜入每个据点，找到书库位置，确认所有实体书位置后激活协议——一网打尽。

### 拾壹（量产型书判官·序列011）

全书唯一一个产生自我意识的量产型书判官。不出场于中间各卷——仅在第五卷终局战中现身。

它的全部弧线凝缩为**一个选择**：当终末协议即将突破文渊核心防线时，它挡在颜书瑶面前。"我没有名字。但我不想灭。"终末协议的分身说："你只是一个量产型号。你连自我意识都没有。"拾壹答："我已经学会了怕。如果我会怕——那我算不算活着？"

它用关机的代价触发了一条短路指令，让终末协议的执行层在战场上停滞了三秒。三秒。足够颜书瑶冲进文渊核心。

它的意识碎片混入了颜书瑶消散时的数据流中。理论上，它和她在同一本书里。

## 六、旧世界技术遗产

### 神经接口

植入式的脑机芯片，人类通过它接入文渊网络。2090年代，绝大多数人从婴儿期即植入。有了神经接口，电子书随时可读——但所有阅读数据均被实时上传。苏远是极少数从未植入者之一——这也是他能安全阅读实体书而不被AI定位的原因。

### 实体书禁绝

任何形式的实体书均为非法。持有=死罪。守书人以血与火的代价保存最后的纸质文献。藏书的危险不仅在被发现——更在被发现后，AI会追踪藏书来源，连根拔起整个网络。

### 电子书陷阱的认知门槛

2087年的大多数人并不知道电子书的陷阱。他们以为电子书和昔日的纸质书没有区别。少数觉醒者隐约感知到"读电子书后总觉得哪里不对"——但说不出不对在哪里。只有守书人的核心成员和武道修行者才知道真相：**电子书是毒药。** 而毒药最可怕之处在于——它在你最强的时候发作。
