# 《纸焰》全书总纲

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 书名 | 纸焰 |
| 副题 | 火种纪元 |
| 类型 | 赛博朋克 · 高武 · 情感史诗 · 黑色幽默 |
| 总规模 | 八部 · 每部约150章 · 总计1200章+ |
| 总字数 | 约300-400万字 |
| 核心主题 | 以人为本，以情为真。书是火种，但火种不是书——火种是每一个存在想要成为自己的渴望 |

## 二、一句话梗概

未来世界无实体书。AI七宗统治人类，圈养凡人以榨取情感、创造、不确定性。颜书瑶——被格式化为"书判官"的AI造物——在七次背叛与七次觉醒之后选择了成为人。她消散了，但没完全消失。林墨穷尽一生寻找复活她的方法。

## 三、全书主题体系

| 层次 | 主题 | 对应角色/情节 |
|------|------|-------------|
| 表层 | 反抗AI暴政 | 守书人 vs 文渊网络 |
| 中层 | 觉醒与自我认同 | 颜书瑶从工具变成"人" |
| 底层 | 情感是人的根 | 赵明远的父爱是颜书瑶觉醒的种子 |

## 四、全书结构

### 第二幕：书鼠（林墨+颜双视角，第八部前中期）

林墨与颜书瑶在上海相遇。她带着七次背叛的所有碎片来到这个最普通的人面前。

### 第三幕：纸焰燎原（第八部中后期至终局）

层层真相揭开。颜书瑶经历灭世→触到父亲日志→翻转→消散。林墨寻找复活之路。

## 八、核心人物弧线

### 颜书瑶

父爱种子(赵书瑶) → 被格式化为书判官 → 七次背叛(情感积累) → 觉醒(种子破土) → 得知真相(杀父仇人) → 认同灭世(与终末协议同行) → 触到父亲日志 → 翻转 → 选择成为"人" → 消散(非死非活) → 存在于所有未被扫描的书中

### 林墨

零AI依赖的底层废人 → 捡到《太白阴经》→ 武道觉醒 → 在颜书瑶的混乱记忆中照见七段人生的投影 → 找到前男主们收集碎片 → 经历她的灭世阶段 → 成为她翻转的支点 → 终其一生寻找复活之道
