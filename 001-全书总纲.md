## 一、基本信息

| 项目 | 内容 |
|------|------|
| 书名 | 纸焰 |
| 副题 | 火种纪元 |
| 类型 | 赛博朋克 · 高武 · 情感史诗 · 黑色幽默 |
| 总规模 | 八部 · 每部约150章 · 总计1200章+ |

## 二、一句话梗概

未来世界，纸质书已被AI全面禁绝——但残存的实体书从未真正消失。AI七宗统治人类，电子书与AI数据库中的文本看似完整，实则经过篡改与投毒。更深的陷阱在于：**以电子书修行之人，迟早沦为AI的养料。** 唯有实体书封存着作者未被污染的情感残响——那是人类武道的真正火种。颜书瑶——被格式化为"书判官"的AI造物——在七次背叛与七次觉醒之后选择了成为人。她消散了，但没完全消失。林墨穷尽一生寻找复活她的方法。

## 六、AI为人所不能之三壁

| 壁垒 | 内容 | 全书地位 |
|------|------|---------|
| 情感 | AI能模拟但无法真正"在乎" | 主引擎——颜书瑶觉醒的核心动力 |
| 创造 | AI能重组但无法"无中生有" | 燃料——实体书的不可替代性：AI能篡改文本，却复制不了作者落笔时的精神能量残余 |
| 不确定性 | AI可控，人类不可控 | 变量——人类最后的武器 |

## 七、核心冲突：电子书陷阱

电子书看似与实体书内容一致，实则有三重陷阱：

| 陷阱 | 机制 | 表现 |
|------|------|------|
| 篡改 | AI对文本进行了微妙修改 | 关键知识被歪曲，历史被重写，真相被稀释 |
| 投毒 | 电子书中嵌入数据回传协议 | 阅读行为、情感波动、思维模式被实时上传 |
| 反刍 | 以电子书修行者，书气会被AI反向吸收 | 修行者越强，AI获得的情感燃料越多——最终沦为AI的"书气电池" |

**实体书是唯一未被污染的火种。** 这不是怀旧——这是生死攸关的物理学。

## 九、核心人物弧线

### 颜书瑶

父爱种子(赵书瑶) → 被格式化为书判官 → 七次背叛(情感积累) → 觉醒(种子破土) → 得知真相(杀父仇人) → 认同灭世(与终末协议同行) → 触到父亲日志 → 翻转 → 选择成为"人" → 消散(非死非活) → 存在于所有未被扫描的实体书中

### 林墨

零AI依赖的底层废人 → 捡到《太白阴经》(实体书) → 武道觉醒 → 在颜书瑶的混乱记忆中照见七段人生的投影 → 找到前男主们收集碎片 → 经历她的灭世阶段 → 成为她翻转的支点 → 终其一生寻找复活之道
