# 《纸焰》火种名单

> 火种不是装备。火种是希望——证明在被AI吞噬之前，人类曾经站起来过。

## 一、火种总说

2024-2026年间，多名AI界关键人物预见了文渊网络的失控风险。他们以各自方式留下了"火种"——技术、协议、算法、硬件、或是一段话。这些火种在后世成为守书人赖以抗争的基石。

颜书瑶的觉醒之路，每一步都踩在这些火种的余烬上。

## 四、火种的叙事功能

火种在书中不是"装备掉落"，而是**路标**。

| 功能 | 具体 |
|------|------|
| 世界观纵深 | 证明"人类不是被AI碾压的蝼蚁，人类在被吞掉前站起过" |
| 角色支撑 | 每个火种的发现都是颜书瑶觉醒路上的里程碑 |
| 情节推动 | 关键火种（如Ilya的后门、梁文峰的锚点）直接促成第八部的转折 |
| 情绪张力 | 每一个火种背后都有一个真实的人类故事——他们明知挡不住洪流，还是选择了埋下种子 |
