## 三、第一卷男主·沈以南

所有男主统一用"关系类型"而非"人格标签"来定义——读者记住的不是"他是黑市王"，而是"他看穿了她"。

### 沈以南——她试图保护的

| 项目 | 内容 |
|------|------|
| 年龄 | 35岁 |
| 身份 | 北京守书人据点领袖 |
| 境界 | 开卷境·巅 |
| 职业审定 | 触知系·识者（辅修：诗号士·残响态） |
| 外貌 | 旧金属框圆眼镜，灰蓝旧棉布衬衫，面容沉静 |
| 核心动作 | 把她递来的《李太白集》放进了胸前口袋——离心最近的位置 |
| 核心台词 | "我不在乎你是谁，我只知道你是你。" |
| 终局 | 死于第一卷。颜书瑶在建国成功前一刻，一掌穿胸。没有火焰，没有倒塌——她亲手杀的。遗物中有一本手抄《诗经》，扉页写着"给书瑶" |
| 全书定位 | 第一粒火种。死后以"似曾相识"形式贯穿全部五卷 |
| 无意识习惯 | 补书时用指腹反复摩挲书页边缘，确认纤维走向后才下笔；总是把修好的书先贴近鼻子闻一下 |

| 全书功能 | 他是颜书瑶"意识种子"的源头。他的日志是她在终局中翻转的关键 |

### 阿蕊

| 项目 | 内容 |
|------|------|
| 年龄 | 7-8岁 |
| 身份 | 北京据点幸存儿童，召唤系修行者（待觉醒） |
| 职业审定 | 工艺系·召唤师（待觉醒）——全守书人体系中断代三代的唯一当代持有者 |
| 外貌 | 瘦小，头发扎两条辫子，左边的总是散了。脸上常带灰——不是脏，是废墟里长大的孩子不会洗脸 |
| 核心动作 | 一直在折纸。折坏了不扔，拆开重折。她的手指比同龄孩子灵巧得多——据点里的人只当她是手巧，不知道这是召唤系的精神力在找出口 |
| 召唤媒介 | 纸鹤——以折叠过程中注入纸脉的精神力为印，以墨焰晶核为驱动能源 |
| 当前状态 | 第一卷尚未觉醒。她折的纸鹤只是纸鹤——每一只都缺最后一道激活。但她从未停止折叠 |
| 第一卷定位 | 七岁，只会折纸的孩子。她的纸鹤在无意中嵌入了精神力印记——但第一卷中它们只是纸鹤。召唤系之觉醒，待后续卷册。 |
| 与颜书瑶的关系 | 她是颜书瑶在全书中第一个主动去"保护"的人——不是任务指令，是她在混凝土碎片落下时用手护住了那朵花。阿蕊不需要知道这件事。她只需要知道"小满姐姐不会走" |
| 核心台词 | "那它是什么都可以。"（对自己的召唤兽说的——一只折坏了但被她收在口袋里的纸鹤） |
| 无意识习惯 | 回答不上问题时，手指会在衣角上反复折一道褶 |
