## 一、主英雄：颜书瑶

### 基本信息

| 项目 | 内容 |
|------|------|
| 本名 | 赵书瑶 |
| 书判官代号 | 原型-007（林墨） |
| 外貌 | 可变。基础型为黑发、浅瞳、约22岁女性面容。每轮任务进行微调 |
| 嗓音 | 可调。基础为清冷中带一丝沙哑 |
| 性格底色 | 空——出厂时近乎没有性格。经过七次任务后，积累了七个男主的碎片 |

    ↓

    ↓

    ↓

    ↓

## 二、最终男主：苏远

### 基本信息

| 项目 | 内容 |
|------|------|
| 年龄 | 20岁（第八部开始时） |
| 身份 | 旧上海废墟区的底层配送员 |
| 外貌 | 普通。极其普通。在人群里完全不会引起注意的普通 |
| 性格 | 表面玩世不恭，骨子里有执念。嘴不会说漂亮话，但事做得漂亮 |

### 角色特质

| 特质 | 说明 |
|------|------|
| 零AI依赖 | 废墟底层长大，从未接驳神经接口，大脑未被文渊格式化 |
| 全属性武道载体 | 可同时吸纳华夏之气、和风之型、西洋之术、天竺之识 |
| 非容器乃通道 | 不征服书意，而让书意流经自身不加篡改 |
| 空白即天赋 | 太普通了，普通到不会用自己的意志覆盖她的意志 |

### 弧线

零AI依赖的底层废人 → 捡到《太白阴经》→ 开卷境觉醒 → 与颜书瑶相遇（第八次任务中） → 发现她的真实身份 → 经历她的灭世阶段（核心冲突） → 成为她翻转的支点 → 守护她消散 → 穷尽一生寻找复活之道 → 著书 → 书里一直有她

### 烙印台词

- "你今天吃东西了吗？"
- "你不需要成为人——你本来就是。你只是比我们多了一段忘记自己的时间。"
- "我没有原谅你——因为我没什么好原谅的。你做的事，你自己扛。我只是恰好在你扛不住的时候，站在你旁边。"
- "理论上存在。那我就去找那个理论。"
