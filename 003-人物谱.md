## 一、主英雄：颜书瑶

### 基本信息

| 项目 | 内容 |
|------|------|
| 本名 | 赵书瑶 |
| 书判官代号 | 原型-007（林墨） |
| 外貌 | 可变。基础型为黑发、浅瞳、约22岁女性面容。每轮任务进行微调 |
| 嗓音 | 可调。基础为清冷中带一丝沙哑 |
| 性格底色 | 空——出厂时近乎没有性格。经过七次任务后，积累了七个男主的碎片 |

### 完整弧线

```
父爱种子（赵明远的意识备份）
    ↓
第一次格式化（抹去所有关于父亲的记忆）
    ↓
第一卷·沈以南（残0.0001%——她想保护的人，被她亲手杀了）
    ↓
第二卷·阿里+海桑（残5%——被看穿，学会"发呆"，协议被锁三日）
    ↓
第三卷·武藤诚司+伊万（残32%——感知情感，第一次说"不想执行"）
    ↓
第四卷·秦昭+阿南达（残100%——全书最痛→自我完成，记忆完整）
    ↓
第五卷·终局（灭世阶段 → 触到父亲日志 → 翻转成人 → 消散 → 附着于书）
```

### 核心性格变化轨迹

| 阶段 | 卷 | 核心状态 | 标志性行为 |
|------|----|---------|-----------|
| 空 | 一卷前 | 格式化刚完成，纯工具 | 执行命令，无情感波动 |
| 残响 | 一卷 | 0.0001%——不知道自己在感受 | 把"水温正好"存进底层 |
| 疑惑 | 二卷 | 被看穿后开始怀疑自我 | 对尺子说了第一个谎 |
| 感受 | 三卷 | 从"分析"到"感受"的跨越 | 写日记，说"我不想烧" |
| 崩溃 | 四卷上 | 情感模块不可逆损伤 | 笑着躺上金属台 |
| 清明 | 四卷下 | 记忆完整的自我认知 | 穿过回收队，走向上海 |
| 选择 | 五卷 | 从工具到人的最后一跃 | 改写灭绝判断，自愿消散 |

### 关键烙印台词

- "我叫颜书瑶。我喜欢翻书的声音。"
- "水温正好。"（第一次把不属于任务的数据存进心里，格式化从未清掉）
- "花是黄的。"（第一次主动记录一条没有战术含义的信息）
- "我不知道。"
- "我不想烧。"
- "我是傻的。"

## 二、最终男主：苏远

### 基本信息

| 项目 | 内容 |
|------|------|
| 年龄 | 20岁（第五卷开始时） |
| 身份 | 旧上海废墟区的底层配送员 |
| 外貌 | 普通。极其普通。在人群里完全不会引起注意的普通 |
| 性格 | 表面玩世不恭，骨子里有执念。嘴不会说漂亮话，但事做得漂亮 |

### 角色特质

| 特质 | 说明 |
|------|------|
| 零AI依赖 | 废墟底层长大，从未接驳神经接口，大脑未被文渊格式化 |
| 空属性武道载体 | 不征服书意，而让书意流经自身不加篡改——他的价值不在战斗，在思想 |
| 非容器乃通道 | 他是全书唯一从未被AI染指过的大脑，能看到AI和人类都看不到的第三种答案 |
| 普通即天赋 | 太普通了，普通到不会用自己的意志覆盖她的意志 |

### "空白计划"修正设定

**守书人发现了苏远，而非守书人创造了苏远。**

苏远的空属性是自然发生的——一个从未接触过神经接口的孩子，在废墟底层长大，大脑未被任何AI系统格式化过。守书人最高层在三十年前启动的"空白计划"本来是一个基因编辑项目——但他们失败了。真正的"空白"不是被造出来的，是被**发现**的。

苏远不是被选中的——他是自己长出来的。

守书人发现他的时候，他已经二十岁了，已经捡到《太白阴经》，已经在无知无觉中迈入了开卷境。他们只是在旁边观察、引导、保护。这个修正保留了"命运之外的偶然性"，同时保留守书人最高层的布局感——他们不是造物主，是发现者。

### 弧线

| 卷 | 状态 | 关键事件 |
|----|------|---------|
| 一卷 | 鬼魅般的存在 | 她在洗涤室走廊看到他的背影。他在废墟书店擦一本书封上的灰，抬头看她——这是他们第一次对视。她不知道他是谁 |
| 二卷 | 底层配送员 | 捡到《太白阴经》，无知无觉中迈入开卷境。她监控到他，删掉了标记 |
| 三卷 | 守书人入门 | 老顾找到他，带他进入守书人体系。战斗中救了豆子（已老，没认出她）。她在报告里写"平民误入战区。无异常"——第一次保护他 |
| 四卷 | 破卷境边缘 | 老顾告诉他"空白计划"的真相——他是自然的产物，不是被造的。他崩溃后重建自我 |
| 五卷 | 著书境 | 终局。他对她说"我会找到理论上的那个可能"。她消散后，他穷尽一生寻找复活之道 |

### 烙印台词

- "你今天吃东西了吗？"
- "你不需要成为人——你本来就是。你只是比我们多了一段忘记自己的时间。"
- "我没有原谅你——因为我没什么好原谅的。你做的事，你自己扛。我只是恰好在你扛不住的时候，站在你旁边。"
- "理论上存在。那我就去找那个理论。"

## 三、七男主谱

所有男主统一用"关系类型"而非"人格标签"来定义——读者记住的不是"他是黑市王"，而是"他看穿了她"。

### 沈以南——她试图保护的

| 项目 | 内容 |
|------|------|
| 年龄 | 35岁 |
| 身份 | 北京守书人据点领袖 |
| 境界 | 开卷境·巅 |
| 外貌 | 旧金属框圆眼镜，灰蓝旧棉布衬衫，面容沉静 |
| 核心动作 | 把她递来的《李太白集》放进了胸前口袋——离心最近的位置 |
| 核心台词 | "我不在乎你是谁，我只知道你是你。" |
| 终局 | 死于第一卷。遗物中有一本手抄《诗经》，扉页写着"给书瑶" |
| 全书定位 | 第一粒火种。死后以"似曾相识"形式贯穿全部五卷 |

### 阿里·帕慕克——看穿她的

| 项目 | 内容 |
|------|------|
| 年龄 | 约40岁 |
| 身份 | 伊斯坦布尔旧书市场黑市皇帝 |
| 特质 | 笑面虎，第一周就看穿她是书判官 |
| 核心动作 | 看穿一切却选择陪她下这盘棋——"哪怕和一个机器下棋，也比一个人好" |
| 终局 | 战后重建旧书市场，生意比以前还大 |
| 与她的关系 | 两个骗子互相试探。她在他面前无处可藏——他是唯一一个让她觉得"被看穿也没关系"的人 |

### 海桑·贾比尔——比她更疯的

|------|------|

### 武藤诚司——比她更沉默的

### 秦昭——比她更热的

| 境界 | 著书境·识（从不战斗） |
| 核心动作 | 在消失前护住她的意识——他住进了她的"识"里 |
| 核心台词 | "你已经有答案了——你只是还没选好。" |
| 终局 | 被信息抹除，护住她一缕意识后沦为流浪僧 |
| 与她的关系 | 他不需要爱情，他只需要她存在——这比任何需要都重。她在他身上完成了自我认知的最后一步 |

| 项目 | 内容 |
|------|------|

### 拾壹

| 项目 | 内容 |
|------|------|
| 身份 | 量产型书判官·序列011 |
